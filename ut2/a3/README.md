#### ***Desarrollo***. <a name="id4"></a>

Para empezar podemos comenzar hablando sobre el origen brevemente. IntelliJ fue creado por JetBrains y su primera version publica fue mostrada ante el publico en 2001. En aquel momento, Java o Eclipse eran bastante toscos por lo que fue una revolucion la aparicion de IntelliJ. Su nombre viene de "Intelligent Java" ya que desde su primera version parecia leer la mente de los trabajadores con sus sugerencias automaticas y refactorización del codigo en tiempo real.
Nacio en principio solo para Java pero hoy en día el IDE soporta una gran variedad de lenguajes como Kotlin, Groovy, SQL, etc.

La instalacion del programa es bastante sencilla. Los pasos que vas a ver los realizamos como se menciono anteriormente en una maquina virtual Linux Ubuntu.
- Lo que vamos a hacer primero es ir a la pagina del software para descargarnos los archivos para nuestro sistema Linux. (Captura 1ª "Descargar_IntelliJ IDEA" en la carpeta /img).
- Posteriormente lo que vamos a hacer es entrar en la terminal y movernos a la carpeta donde hayamos descargado los archivos. Cuando estemos dentro de esta carpeta nos moveremos una vez más a la carpeta que nos hemos descargado. Dentro tendremos el archivo que tendremos que instalar, lo cual haremos desde la terminal. (Captura 2ª "Instalacion_IntelliJ IDEA" en la carpeta /img).
- Por ultimo lo quer tendremos que hacer es salir a la localizacion donde hayamos instalado el programa final y ya podremos acceder dentro del software (Captura 3ª "IntelliJ_Instalado" en la carpeta /img).



> ***IMPORTANTE:*** si estamos capturando una terminal no hace falta capturar todo el escritorio y es importante que se vea el nombre de usuario.
